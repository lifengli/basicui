## Dev

### 1. start uiserver at :3001

> npm start

### 2. start basicui in dev mode

> npm start

*by default, the http server binds to localhost or 0.0.0.0 at port 8080

### 3. Test

> npm run lint

> npm run test

> npm run coverage

## Build

> npm run build

*bundle will be generated to /public/javascrips/ on uiserver
